# work_unity（类魂动作游戏）

本项目为一款 **类黑魂（Souls-like）第三人称动作游戏**，  
使用 **Unity 2022.3 LTS** 开发，由两人协作完成。

---

## 一、Unity 版本
- Unity **2022.3.xf1 LTS**
- 所有成员必须使用 **相同 Unity 版本**

---

## 二、项目目录说明（Assets）

```text
Assets/
├─ Art/            # 模型 / 贴图 / 材质（纯美术资源）
├─ Animations/     # 动画 / Animator Controller / Mask
├─ Scenes/         # 所有 Scene（.unity）
├─ Prefabs/        # 预制体（玩家 / 敌人 / UI / 道具）
├─ Scripts/        # 所有 C# 脚本
└─ Audio/          # 音效 / BGM
